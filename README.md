# dm1trr1y-page

A personal web page showcasing projects and portfolio.

## Tech Stack

### Frontend

- **HTML5** - Markup structure
- **CSS3** - Styling and animations (loader.css, style.css)
- **JavaScript (ES6+)** - Interactive functionality

### Build Tool

- **Vite** - Fast build tool and development server

### Libraries & Dependencies

#### UI & Interactions

- **Swiper** - Touch-enabled carousel/slider component
- **Spin.js** - Animated loading spinner

#### HTTP Client

- **Axios** - Promise-based HTTP client for API requests

### Project Structure

```
src/
├── css/
│   ├── loader.css        # Loading spinner styles
│   └── style.css         # Main stylesheet
├── img/                  # Image assets
└── js/
    ├── api.js            # API request handling
    ├── loader.js         # Loading spinner logic
    ├── main.js           # Application entry point
    ├── swiper.js         # Carousel initialization
    └── textRotation.js   # Text rotation effects
```
